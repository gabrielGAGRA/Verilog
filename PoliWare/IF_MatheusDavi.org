* Especificação 
 -  Um registrador (Program Counter) que tem uma entrada e uma saída de 32 bits, sensível a borda de subida do clock e com reset.
 -  Um módulo para somar 4 na saida do Program Counter e devolver o novo valor na entrada do PC.
 -  Uma memória para as instruções (instruction memory) , 64 registradores de 32 bits concatenados, sensível a borda de subidad do clock e com reset.

* Módulos 
 Três módulos devem ser feitos, um para cada componente de acordo com as tabelas abaixo:
** Program Counter

   | Nome      | Tipo         | Tamanho(bits) | Descrição                                    |
   |-----------+--------------+---------------+----------------------------------------------|
   | PC_input  | input        |            32 | endereço de input                            |
   | clk       | input        |             1 | sinal de clock                               |
   | reset     | input        |             1 | sinal de reset para zerar o PC               |
   | PC_output | output (reg) |            32 | endereço de output para a instruction memory |
   |-----------+--------------+---------------+----------------------------------------------|

** Módulo de soma

    | Nome         | Tipo   | Tamanho(bits) | Descrição                |
    |--------------+--------+---------------+--------------------------|
    | adder_input  | input  |            32 | input que recebe do PC   |
    | adder_output | output |            32 | output que retorna ao PC |
    |--------------+--------+---------------+--------------------------|
  
** Instruction Memory
   | Nome             | Tipo         | Tamanho(bits) | Descrição                                         |
   |------------------+--------------+---------------+---------------------------------------------------|
   | read_address     | input        |            32 | lê o endereço do PC                               |
   | clk              | input        |             1 | sinal de clock                                    |
   | reset            | input        |             1 | sinal de reset para zerar as instruções           |
   | instruction_list | reg          |       32 [64] | vetor de registradores para guardar as instruções |
   | instruction_out  | output (reg) |            32 | qual instrução sai para o resto do processador    |
   |------------------+--------------+---------------+---------------------------------------------------|
